# Details
* Font Used : Iosevka Custom
* GTK Theme : Generated by [warnai](https://github.com/reorr/warnai)
* Icons     : Custom Version of papirus
* OS        : Arch Linux (Btw)
* WM        : BSPWM
* Terminal  : Kitty, Urxvt
* Shell     : zsh
* Bar       : Polybar
* Lockscreen: betterlockscreen
